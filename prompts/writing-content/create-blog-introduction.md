# Create Blog Introduction

**Category:** Writing & Content  
**Source:** Glean

## Prompt

> Create long/short form copy based on existing assets/materials.
> Instructions:
> 1. Read the provided document or URL.
> 2. Extract the core message and audience.
> 3. Write a compelling intro (100–150 words) that hooks the reader immediately.
> 4. Use the first sentence to state the problem or promise.
> 5. End the intro with a natural bridge to the rest of the article.
