# Summarize Recent Changes to MR/Issue

**Category:** Coding & Development  
**Source:** GitLab Duo

## Prompt

> Summarize recent activity on this MR/Issue so I can get caught up quickly:
> 
> MR/Issue: [LINK or PASTE CONTENT]
> I was last active: [DATE] or "I'm coming in fresh"
> 
> ## Executive Summary (30 seconds)
> What's the current status? What's the single most important thing I need to know right now?
> 
> ## What's Changed Since [DATE]
> Timeline of significant events:
> - [Date]: [What happened — decision made, code changed, blocker found]
> - [Date]: [...]
> 
> ## Key Decisions Made
> What was debated and what was decided?
> - Decision: [X] — Reason: [Y] — Decided by: [Z]
> 
> ## Current Blockers
> What's preventing progress right now?
> - Blocker: [description] — Owner: [who needs to act] — ETA: [if known]
> 
> ## My Action Items
> Based on the current state, what do I need to do?
> 1. [Action]
> 2. [Action]
> 
> ## Context I Need
> What background do I need to contribute meaningfully to this MR/Issue?
