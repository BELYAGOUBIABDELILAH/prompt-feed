# Adjust Code to Match Style Guides

**Category:** Coding & Development  
**Source:** Glean

## Prompt

> Review a code snippet and suggest improvements to align it with specified style guidelines.
> For each deviation from the style guide:
> 1. Quote the problematic line
> 2. Explain which rule it violates
> 3. Show the corrected version
> 4. If multiple valid approaches exist, note which the team prefers and why
> Output the full corrected snippet at the end, ready to copy-paste.
