# Get Started Writing Code

**Category:** Coding & Development  
**Source:** Glean

## Prompt

> Find relevant guides, documents, or code files to begin writing code.
> Given the task I'm starting:
> 1. Search the codebase for similar implementations or patterns
> 2. Identify the relevant files, modules, and services I'll need to touch
> 3. List any existing utilities, helpers, or abstractions I should reuse
> 4. Highlight the team's conventions for this type of code (naming, structure, testing)
> 5. Surface any known gotchas, deprecated patterns, or things to avoid
> Output: A quick-start context block I can use before writing a single line.
