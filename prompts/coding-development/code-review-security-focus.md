# Code Review (Security Focus)

**Category:** Coding & Development  
**Source:** Glean

## Prompt

> Identify security issues and suggest ways to mitigate them.
> Review the following code for:
> - SQL injection vulnerabilities
> - XSS attack surfaces
> - Insecure authentication or session handling
> - Hardcoded secrets or credentials
> - Improper input validation
> - Exposed error messages with internal details
> For each issue found: state severity (Critical / High / Medium / Low), explain the risk, and provide a corrected code snippet.
