**Category:** Coding & Development  
**Source:** GitLab Duo

## Prompt

> Help me estimate the implementation effort for:
> 
> Feature/Task: [DESCRIPTION]
> Team: [N developers, skill level: junior/mid/senior mix]
> 
> ## Work Breakdown Structure
> 
> Break the work into tasks:
> 
> | # | Task | Type | Complexity | Estimate |
> |---|---|---|---|---|
> | 1 | [Task] | [Backend/Frontend/DB/Infra/Testing] | [Low/Med/High] | [hours] |
> 
> **Complexity guide:**
> - Low: well-understood pattern, similar to existing code
> - Medium: some unknowns, requires some design
> - High: significant unknowns or first time doing this
