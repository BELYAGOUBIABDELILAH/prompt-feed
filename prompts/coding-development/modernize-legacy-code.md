# Modernize Legacy Code

**Category:** Coding & Development  
**Source:** GitLab Duo

## Prompt

> Modernize the following legacy code to use current best practices:
> 
> [PASTE LEGACY CODE]
> 
> Language/Framework: [SPECIFY]
> Target version: [SPECIFY e.g., Python 3.12, ES2024, Java 21]
> 
> ## Modernization Checklist
> Identify and fix each of these if present:
> - Deprecated APIs or syntax
> - Callback hell → async/await or promises
> - Manual resource management → using/with statements
> - Old-style string formatting → template literals / f-strings
> - var → const/let (JavaScript) or equivalent
> - Mutation-heavy code → immutable patterns
> - Missing type hints/annotations
> - Null safety improvements
> 
> ## Modernized Code
> Provide the complete updated version with inline comments explaining each significant change.
> 
> ## Migration Notes
> - Breaking changes introduced (if any)
> - Dependencies to update (package names and versions)
> - Tests that need updating
> - Estimated performance improvement
