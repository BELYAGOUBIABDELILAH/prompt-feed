# Build Full-Stack Application

**Category:** Coding & Development  
**Source:** GitLab Duo

## Prompt

> Scaffold a full-stack application with the following requirements:
> 
> Application name: [NAME]
> Purpose: [WHAT IT DOES]
> Tech stack:
>   - Frontend: [React / Vue / Next.js / etc.]
>   - Backend: [Node.js / Python / Go / etc.]
>   - Database: [PostgreSQL / MySQL / MongoDB / etc.]
>   - Auth: [JWT / OAuth / Session]
> 
> ## Project Structure
> Generate the complete directory structure first, then implement each section.
> 
> ## Database Schema
> Design the schema for core entities:
> - Tables/collections with fields and types
> - Relationships and foreign keys
> - Indexes for common queries
> - Migration file
> 
> ## API Layer
> Design RESTful (or GraphQL) API:
> - Endpoints with HTTP methods
> - Request/response schemas
> - Authentication middleware
> - Error response format
> - Input validation
> 
> ## Frontend Components
> Core components to scaffold:
> - Layout/navigation
> - Auth forms (login/signup)
> - Main feature view
> - API integration layer (fetch/axios wrapper)
> 
> ## Development Setup
> - docker-compose.yml for local development
> - Environment variables needed (.env.example)
> - README with setup steps (< 5 steps to running locally)
> 
> Provide complete, working code for each section.
