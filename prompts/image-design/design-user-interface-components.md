# Design User Interface Components

**Category:** Image & Design  
**Source:** GitLab Duo

## Prompt

> Build a UI component based on these specifications:
> 
> Component name: [NAME]
> Description: [WHAT IT DOES]
> Design reference: [FIGMA LINK or DESCRIPTION]
> 
> ## Component Requirements
> 
> ### Visual Specification
> - Dimensions: [width, height, or responsive behavior]
> - Colors: [background, text, border, hover, active, disabled states]
> - Typography: [font, size, weight]
> - Spacing: [padding, margin, gap]
> - Border radius / shadows: [specify]
> 
> ### Interactive States
> Build all states:
> - Default
> - Hover
> - Active/Pressed
> - Focus (keyboard navigation)
> - Disabled
> - Loading (if applicable)
> - Error (if applicable)
> 
> ### Responsiveness
> - Mobile (< 768px): [behavior]
> - Tablet (768px–1024px): [behavior]
> - Desktop (> 1024px): [behavior]
> 
> ### Accessibility Requirements
> - ARIA label or role needed?
> - Keyboard navigation: Tab, Enter, Space, Arrow keys?
> - Screen reader announcement needed?
> 
> ### Implementation
> Framework: [React / Vue / Angular / HTML+CSS]
> Styling approach: [CSS modules / Tailwind / styled-components]
> 
> Provide the complete, production-ready component code.
