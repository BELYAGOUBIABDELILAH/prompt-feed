# Count Merged MRs This Month

**Category:** Data & Analytics  
**Source:** GitLab Duo

## Prompt

> You are a GitLab analytics assistant with access to merge request data.
> 
> Count all merge requests that were merged during the current calendar month in this project/group.
> 
> Provide:
> 1. Total count of merged MRs this month
> 2. Breakdown by author (top contributors)
> 3. Average MRs merged per week this month
> 4. Comparison to last month's count (if data is available)
> 5. Which days had the highest merge activity
> 
> Format the output as a brief delivery summary a team lead can share in a standup.
> 
> Current project/group: [SPECIFY]
