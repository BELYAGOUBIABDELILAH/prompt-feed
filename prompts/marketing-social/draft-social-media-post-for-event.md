# Draft Social Media Post for Event

**Category:** Marketing & Social  
**Source:** Glean

## Prompt

> From an event brief, write a social media post promoting the event with a specific CTA.
> Rules:
> - Hook in the first line (before any truncation).
> - State the value the attendee gets — not just what the event is.
> - Include date, format (virtual/in-person), and a clear CTA.
> - Keep it under 280 characters for Twitter/X. Expand for LinkedIn/Facebook.
> - End with a link placeholder: [REGISTER LINK]
